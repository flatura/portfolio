## MyLovelyGarden

**Тип:** B2B SaaS / botanical domain  
**Роль:** Co-founder, System Designer, Backend-oriented Engineer  
**Статус:** active

### Контекст
Платформа для ботанических организаций и коллекций растений. Проектируется как прикладная B2B-система с учетом предметной сложности, ролевой модели доступа и долгосрочной эволюции продукта.

### Зона ответственности
- Проработка доменной модели и ключевых системных сценариев
- Проектирование backend-архитектуры и структуры данных
- Подготовка технической документации и системных решений

- Практическая реализация backend-части и координация общей технической структуры проекта

### Ключевые решения
- Soft multi-tenancy
- Context-aware RBAC
- Гибридная доменная модель: текущее состояние сущностей + append-only история событий для трассируемости и цифровых паспортов растений
- Разделение ответственности между прикладной моделью, словарным контуром и операционными сценариями
