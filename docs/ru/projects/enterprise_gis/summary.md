# Краткое описание

## Статус

опытно-промышленная эксплуатация для внутреннего B2E-сценария; pre-sale архитектура для B2B-продукта

## Роль

системный аналитик / проектировщик решения

## Стек

Домен: Enterprise GIS, логистика, железнодорожная инфраструктура; NDA: детальная архитектура, модели данных и исходный код являются конфиденциальными

## Ценность проекта

Enterprise GIS-платформа для логистического сектора - визуализация корпоративных операционных данных на карте с пространственной привязкой к железнодорожной инфраструктуре.
