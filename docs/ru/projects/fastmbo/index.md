# FastMBO - внутренняя система целеполагания и расчёта премий

**Тип:** Internal automation / MBO system / lightweight internal ERP  
**Роль:** инициатор, системный аналитик, автор концепции, разработчик прототипа  
**Статус:** historical internal project / reconstructed case study  
**Масштаб:** внутреннее использование в организации до 350 сотрудников  
**Период:** ранний этап перехода от ИТ-администрирования к системному анализу и разработке внутренних систем

FastMBO - внутренняя система для автоматизации процесса целеполагания, сбора оценочных листов, согласования достижений и расчёта премий по MBO-модели.
