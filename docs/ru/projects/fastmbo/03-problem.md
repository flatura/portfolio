# Проблема

Основные проблемы:

- высокая трудоёмкость подготовки и обработки оценочных листов;
- риск ошибок при переносе данных;
- сложность контроля статусов заполнения и согласования;
- отсутствие единой цифровой истории по периодам оценки;
- слабая прозрачность процесса для участников;
- зависимость от отдельных сотрудников, знающих порядок ручного расчёта.
