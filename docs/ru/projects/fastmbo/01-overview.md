# Обзор

FastMBO - внутренняя система для автоматизации процесса целеполагания, сбора оценочных листов, согласования достижений и расчёта премий по MBO-модели.

Проект возник из реальной операционной боли: процесс оценки сотрудников выполнялся вручную через бумажные бланки, Excel, повторные проверки, ручную оцифровку и перенос итогов в учётные системы.

Идея заключалась в том, чтобы заменить хрупкий ручной процесс небольшой внутренней ERP-like системой: сотрудники заполняют оценочные листы онлайн, руководители согласуют достижения, ответственные лица управляют периодами, а система рассчитывает баллы, цену балла и итоговые премии.
