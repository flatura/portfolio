# Обзор

## Обзор продукта

**Executive Decision Intelligence Platform PoC** — прототип корпоративной системы управленческой аналитики поверх цифровых следов компании.

Продуктовая идея: руководитель или владелец бизнес-домена задаёт вопрос в свободной форме, а система помогает проверить гипотезу, найти отклонение, связать метрики с документами и подготовить управленческий вывод.

Это не "ИИ-директор", который принимает решения вместо человека. Это слой аналитической поддержки в решениях, который:

- работает с корпоративными данными через контролируемые инструменты;
- показывает ограничения анализа;
- сохраняет запуски инструментов, логи запроса;
- формирует проверяемый ответ с доказательствами.

## Продуктовые режимы

1. **Executive Chat**
   Пользователь задаёт вопрос в свободной форме: "Почему просела маржа?", "Почему time-to-market нестабилен?", "Есть ли риск невыполнения плана?" и получает ответ от системы.

## Технологический стек

| Слой | Решение в PoC | Назначение |
|---|---|---|
| Chat UI | Open WebUI | Временный интерфейс для демонстрации |
| Agent runtime | FastAPI + LangGraph | Stateful диагностический процесс, ветвления, формирование доступных инструментов |
| Tool execution | FastAPI tool-server | Каталог контролируемых инструментов, валидация, структурированный вывод |
| Структурированные данные | PostgreSQL | Финансы, продажи, delivery, ITSM, PMO, встречи, корпоративные данные |
| Документное хранилище | MinIO + Qdrant | Сырые/нормализованные документы, чанки, векторный поиск, ссылки на источники |
| Runtime/cache | Redis | Поддержка рантайма, очереди/кеш где применимо |
| LLM | OpenAI-совместимое API | Планирование, выбор инструментов, интерпретация, синтез, формулировки; не источник истины |
| Infra | Docker Compose | Воспроизводимый лабораторный жизненный цикл, независимый от железа |
