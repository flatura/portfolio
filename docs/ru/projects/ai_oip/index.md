# AI Operational Intelligence Platform

**Тип:** Enterprise AI / система поддержки принятия решений / MVP агентной аналитики  
**Роль:** Solution Architect, System Designer, AI-assisted Prototype Engineer  
**Статус:** рабочий MVP-прототип, июнь 2026

Enterprise AI-прототип для управленческой аналитики на проверяемых данных - **AI Executive Analyst with verifiable evidence**, который помогает руководителям исследовать бизнес-вопросы без непрозрачного AI-рассуждения.
