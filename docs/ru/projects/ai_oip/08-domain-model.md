# Доменная модель

## Синтетический enterprise dataset

Подготовил синтетические enterprise demo data для fashion/retail/manufacturing-компании, включая связанные бизнес-домены:

* **Financial** - margin, revenue, discounts, COGS, product mix
* **Delivery** - roadmap и delivery metrics
* **ITSM** - service и incident context
* **PMO** - project и portfolio management data
* **Meetings** - decisions и meeting notes
* **Documents** - document evidence для RAG
* **Scenario-based anomalies** - cross-domain diagnostic scenarios
