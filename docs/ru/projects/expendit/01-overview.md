# Обзор

## Краткое описание

Создать лёгкую внутреннюю ERP-систему для отдела ИТ: не как большой корпоративной платформы, а как прикладного инструмента, который помогает видеть текущие остатки, историю движения материалов и будущую потребность в закупках.

## Технологический стек

* Spring Boot backend
* Thymeleaf простой UI
* Локальный запуск
