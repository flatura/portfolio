# Краткое описание

## Статус

MVP / подготовка к пилотному запуску

## Роль

сооснователь, системный проектировщик, backend-oriented engineer

## Стек

Java 21, Spring Boot, PostgreSQL/PostGIS, MinIO/S3-compatible storage, Angular, OpenLayers, Docker, LLM

## Ценность проекта

Botanical SaaS - мультитенантная SaaS-платформа для ботанических садов, питомников, селекционеров, научных коллекций и частных собраний растений.

## Что демонстрирует

Этот проект демонстрирует мои способности:

* работать на стыке системного анализа, backend дизайна, имплементации и деплоя.
* довести идею до запуска системы с несколькими зонами доступности и дорожной картой масштабирования
* стратегически планировать имплементацию SaaS малого-среднего масштаба
* безопасно и контролируемо применять LLM-генерацию кода для имплементации фичей
