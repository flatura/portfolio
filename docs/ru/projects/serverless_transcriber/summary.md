# Краткое описание
## Статус
В эксплуатации

## Роль
Системный проектировщик

## Стек
Python, AWS Lambda, API Gateway, DynamoDB, AWS S3, Terraform, LLM API integration

## Ценность проекта
Для пользователя:

- быстрая транскрибация длинных аудиозаписей без загрузки в тяжёлые SaaS-сервисы;
- контролируемый доступ к результатам через Cognito и предподписанные ссылки;
- минимальные постоянные расходы за счёт serverless-модели.

Для профессионального профиля:

- практическое проектирование AWS serverless workflow;
- работа с асинхронной обработкой, webhook, polling UI и жизненным циклом задачи;
- применение Infrastructure-as-Code через Terraform;
- выбор решений исходя из эпизодической нагрузки и ограничений стоимости (cost-driven architecture).
