# Контекст и проблема

## Контекст и предпосылки

- Есть эпизодическая необходимость в быстрой и дешевой транскрибации больших аудио-файлов (1.5ч+) с минимальными затратами на инфраструктуру.
- Нужна практика работы с облачными сервисами Amazon

## Проблема

- Сервисы платной транскрибации имеют избыточную функциональность и неоптимальную стоимость.
- Некоторые записи содержат приватные сведения, которые не хотелось бы кому-то отдавать
