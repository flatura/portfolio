# Компромиссы

## Serverless vs постоянный хостинг

* **VPS не подходят:** периодическая оплата мощностей даже когда сервис не используется; плохо для эпизодической нагрузки.
* **Telegram bot не подходит:** ограничения на размер аудио-файла; логику бота всё равно нужно где-то размещать.

## Управляемый API транскрибации vs self-hosted модели

* **Бюджет на GPU/хостинг отсутствует** - запуск opensource модели на своих мощностях не предполагается.
* **Сторонний API (AssemblyAI)** - оптимальное соотношение качество/цена; ограничение на 5 одновременных процессов транскрибации.
