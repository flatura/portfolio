## Флагманский проект: SaaS ERP для ботанических организаций (MyLovelyGarden)
*Статус: Pre-launch (Подготовка к пилоту), задеплоено на production VPS.*

Комплексная ERP-система для ботанических садов, питомников и селекционеров. Решает проблему стандартизированного учета биологических фондов (более 1.5 млн таксонов) и управления коллекциями. Проект прошел путь от монолитного прототипа до современного decoupled-решения с помощью AI-ассистированной разработки.

* **Бизнес-проблема:** Отсутствие единого стандарта учёта растений и невозможность сквозного обмена данными между организациями.
* **Архитектурная эволюция:**
  * **AS-IS (Итерация 1):** Монолит с Server-Side Rendering (Java Spring Boot + Thymeleaf), высокая связность бизнес-логики и UI.
  * **TO-BE (Текущая MVP-версия):** Разделение на stateless Backend API (Spring Boot) и Frontend SPA (Angular). Полный рефакторинг схемы БД под требования нормализации.
  * **Roadmap (Итерация 3):** Миграция в AWS (ECS, RDS, S3, CloudFront).
* **Ключевые решения (ADR):**
  * **AI-Augmented Refactoring:** Использование связки Cursor + Claude для миграции логики и полного переписывания фронтенда на Angular за 2 недели (кратное сокращение TTM).
  * **Infrastructure:** Контейнеризация (Docker), CI/CD пайплайн (GitHub Actions) для автоматической сборки образов и zero-downtime деплоя.
  * **Data & Security:** Единый валидируемый справочник таксонов, развитая RBAC-модель доступа к данным.
* **Стек:** Java, Spring Boot, Angular, PostgreSQL, MiniO, Docker, OpenLayers, GitHub Actions.
