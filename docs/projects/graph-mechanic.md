# GraphMechanic

## Обзор
Веб-сервис для визуального создания и редактирования графов с привязкой к географическими объектами

### Роль и вклад
Инициатор и автор концепции. Системный аналитик и архитектор: БТ/ФТ/НФТ, C4, ERD, BPMN, ADR, docs-as-code.
