# FMCG Supply Chain Optimization (Demand Forecasting & Planning)
*Industry: FMCG / Machine Learning & Advanced Analytics (2025 – 2026)*

Designed the system landscape, data models, and integration contracts for a demand forecasting (ML) and production planning (Gurobi Solver) system. Acted as a System Designer, bridging the gap between business requirements and Data Architecture.

* **Demand Forecast ML Module:**
  * Conducted business process modeling (AS-IS / TO-BE) and formalized business and functional requirements (BRD/FRD).
  * Conducted a Build vs. Buy analysis, providing technical justification for the in-house development scenario to ensure the required system flexibility.
  * Visualized the target solution architecture by preparing a comprehensive suite of C4 model diagrams (Context, Container, Component) and Data Flow Diagrams (DFD).
  * Designed ETL integration contracts for data preparation to feed ML models (Feature Engineering).

* **Production Planning Module (Gurobi Solver):**
  * Designed the relational data model, preparing DDL scripts for the Prototype and MVP stages.
  * Developed API contracts for backend service interactions.
  * Designed ETL data contracts for seamless data transfer into the mathematical optimization solver.

* **Result:** Delivered a complete package of project and integration documentation, enabling business stakeholders and Enterprise Architects to transparently evaluate the complexity of in-house development and make the final product decision.
