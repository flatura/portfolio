# Domain Model

## Synthetic enterprise dataset

Prepared synthetic enterprise demo data for a fashion/retail/manufacturing company, including connected business domains:

* **Financial** - margin, revenue, discounts, COGS, product mix
* **Delivery** - roadmap and delivery metrics
* **ITSM** - service and incident context
* **PMO** - project and portfolio management data
* **Meetings** - decisions and meeting notes
* **Documents** - document evidence for RAG
* **Scenario-based anomalies** - cross-domain diagnostic scenarios
